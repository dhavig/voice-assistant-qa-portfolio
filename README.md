# Voice Assistant QA Portfolio

QA portfolio project for voice-assistant testing with exploratory charters, adversarial scenarios, bug triage, and release-readiness artifacts.

## Project Overview

This repository contains a portfolio project focused on quality engineering for conversational, voice-assistant style experiences. It demonstrates:

- exploratory testing
- adversarial testing
- customer-impact analysis
- bug triage
- release-readiness thinking
- iPhone and iPad-style workflow validation

## Demo Coverage

The demo includes scenarios around:

- vague reminders
- context carryover
- missing calendar details
- multi-timer state handling
- permissions failures
- unsupported request fallback
- network-offline behavior
- entity disambiguation

## Repository Purpose

This project is intended as a QA portfolio artifact for recruiter, hiring-manager, and interview discussions around customer-facing product quality.
